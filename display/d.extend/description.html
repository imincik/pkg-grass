DESCRIPTION

d.extend can be used to display all objects in currently displayed maps.

NOTES

If no raster map is displayed on a monitor, the default region resolution is calculated to fill window by 512 * 512 cells. Otherwise, it will use the minimum resolution of displayed raster maps.
